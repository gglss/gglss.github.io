## 下载jar包

点击[moke](https://repo1.maven.org/maven2/com/github/dreamhead/moco-runner/)选择版本进行下载，我下载的的是`moco-runner-0.11.0-standalone.jar`

## 使用命令启动测试脚本

```json
java -jar jar包的存放路径 http -p 端口 -c json文件路径
```
