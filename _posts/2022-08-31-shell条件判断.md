##### 基本语法

[ condition ] （<font color='red' font-size='40px'>注意，condition 前后要有空格</font>）

注意：条件非空即为 true,[ atguigu ]返回 true, [] 返回 false.

##### 常用条件判断

###### 两个整数之间进行比较

```shell
= 字符串比较
-lt 小于
-le 小于等于
-eq 等于
-gt 大于
-ge 大于等于
-ne 不等于
```

举个栗子

```shell
[23 -lt 80]
echo $?  # 判断上一步操作是不是正确
```

###### 按照文件权限进行判断

```shell
-r 读
-w 写
-x 执行
```

举个栗子

```shell
[ -w 1.sh ]
echo $?
```

###### 文件类型判断

```shell
-f 文件存在且是一个常规文件
-e 文件存在
-d 文件存在且是一个目录
```

举个栗子

```shell
[ -e 2.sh ]
echo $?
```
