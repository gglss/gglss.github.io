#### 重定向的测试接口
+ 使用了`redirectTo`：目标地址

```json
[
  {
    "description": "重定向到百度",
    "request": {
      "uri": "/redirect"
    },
    "redirectTo": "http://www.baidu.com"
  },
  {
    "description": "重定向到自己的网页上",
    "request": {
      "uri": "/redirect/topath"
    },
    "redirectTo": "/redirect/new"
  },
  {
    "description": "这是被重定到的请求",
    "request": {
      "uri": "/redirect/new"
  },
    "response": {
      "text": "重定向成功了"
    }
  }
]
```
